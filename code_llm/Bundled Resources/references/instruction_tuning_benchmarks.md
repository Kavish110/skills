Instruction tuning & benchmark papers

- FLAN: https://arxiv.org/abs/2210.11416
- Alpaca-style instruction tuning: https://crfm.stanford.edu/2023/03/13/alpaca.html
- Summary: Instruction-tuning datasets improve model helpfulness and generalization by teaching models to follow natural-language instructions.
- What worked: Diverse instruction data and clear response targets improve downstream behavior.
- What didn't: Poorly curated or noisy instructions can create unstable or unhelpful model behavior.
