Scaling Laws (Kaplan et al., 2020)

- ArXiv: https://arxiv.org/abs/2001.08361
- Summary: This paper established empirical scaling laws for neural language models, quantifying how test loss decreases with model size, dataset size, and compute.
- What worked: It provided a predictable planning framework for allocating compute between model size and training data.
- What didn't: The results show that simply increasing model size without increasing data is inefficient; the best performance depends on a compute-optimal balance.
